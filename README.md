# Secure File Integrity Verification System

A cybersecurity-focused web application designed to detect unauthorized file modifications using SHA-256 hashing and secure authentication mechanisms.

## Features
- File integrity verification using SHA-256 hashing
- Tamper detection through hash comparison
- Secure user authentication
- Responsive frontend interface
- Real-time verification workflow

## Tech Stack
- Frontend: React, Vite
- Backend: Flask, Python
- Authentication: JWT
- Security: SHA-256 Hashing

## Project Status
Currently under development with frontend deployment completed and backend integration in progress.

## Future Enhancements
- Blockchain-based verification records
- Cloud database integration
- File history tracking
- Advanced security monitoring
